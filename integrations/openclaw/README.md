# HomeFin OpenClaw Bundle

This bundle ships the HomeFin finance skill, a local MCP server definition, and example workspace settings.

- `skills/homefin/SKILL.md`: finance assistant rules for grounded answers.
- `.mcp.json`: stdio MCP wiring for the HomeFin Python server.
- `agents/finance-agent.md`: descriptive agent profile for operators.
- `commands/` and `hooks/`: placeholders for workspace-level automation prompts and hook packs.
